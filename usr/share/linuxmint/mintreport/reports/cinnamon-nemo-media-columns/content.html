When both thumbnail previews and the nemo-media-columns extension are enable in Nemo, the file browser can get into a state where it loads directory contents indefinitely.

When this happens, the workaround is to kill nemo with the command:

killall nemo

Until this issue is fixed, we recommend to remove nemo-media-columns or to disable it in Nemo's plugins preferences.

apt remove nemo-media-columns
